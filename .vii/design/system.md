## Decision Log

### Background Color — `#FFF5EF` (warm off-white)
**Source:** wgb.agency (`rgb(255,245,239)`) — confirmed as direction by juanmora.co's similar cream base.
**Why not pure white (`#FFFFFF`):** Pure white reads as generic SaaS. The warm tint creates instant distinctiveness.
**Why not dark:** All three references use a light base. Vibhanshu's chosen sites lean warm/light, so the plan's earlier dark suggestion is overridden by his own taste signal.

### Accent Color — `#0028FF` (electric blue)
**Source:** wgb.agency (`rgb(0,40,255)`) + juanmora.co (`#2B4BFF`) — both land on saturated electric blue.
**Why this over the earlier plan's `#e8ff00`:** The references show blue, not yellow. Following references, not speculation.
**Usage rule:** Used sparingly at full saturation — CTA buttons, active states, badges. Not used on body text.

### Secondary Accent — `#FF5A00` (orange)
**Source:** wgb.agency. Provides warm energy for CTAs and hover states without competing with blue.

### Font — Switzer
**Source:** wgb.agency (computed font-family: `Switzer, Arial, sans-serif`).
**Why Switzer:** Geometric sans with humanist touches — professional but not corporate. Free via Fontshare. Renders cleanly at both 84px (hero) and 14px (labels).
**Load via:** `https://api.fontshare.com/v2/css?f[]=switzer@400,500,700&display=swap` — add as `<link>` in `layout.tsx` `<head>`.
**Deviation:** Single font family only (not using cafebinocle's Caprasimo display font — too playful for a founder-facing portfolio).

### Heading Weight — 500 (medium)
**Source:** wgb.agency uses `font-weight: 500` for ALL heading levels (H1 at 84px through H3 at 24px).
**Why:** Elegance via size contrast, not weight contrast. Weight 700 headings read as aggressive; 500 reads as confident.

### Letter-Spacing — `-0.04em` on all headings
**Source:** wgb.agency (`-3.36px` at `84px` = roughly `-0.04em`).
**Why:** Tight tracking at large sizes creates editorial print magazine feel. Standard browser tracking at large sizes looks clunky.

### Type Scale — Major Third (1.25×)
**Base unit:** `1rem = 16px`. Scale: `12 / 14 / 16 / 20 / 24 / 32 / 48 / 68 / 84px`.
**Why Major Third:** Not too steep (avoids huge jumps), not too flat (creates clear hierarchy).

### Spacing — 8px base
**Source:** wgb uses `96px` section padding (= 12 × 8px). cafebinocle uses `80px` section gaps.
**Section gap:** `128px` (--space-32) between major sections — generous whitespace signals confidence.

### Border Radius — `6px` default
**Source:** wgb.agency (`border-radius: 6px` on cards, `3px` on tags).
**Pills (`9999px`):** For tags and badge chips only — from juanmora.co.
**Deviation from cafebinocle:** Its `50px` pill buttons would look too playful/consumer. Portfolio uses `6px` buttons.

### Structural Borders — `1px solid #0A0A0A`
**Source:** wgb.agency (full-width `1px` black horizontal rules between every section).
**Usage:** Section dividers, not card borders. Creates editorial newspaper grid feel.

## Base Spacing Unit

**8px.** All spacing tokens are multiples of 4px for granularity, 8px for rhythm. `--space-4 = 16px` is the atomic content padding unit.

## Type Scale Ratio

**Major Third (1.25×)** starting from `16px` base.

---

## What NOT to Do

- Never use `font-weight: 700` on headings — use size contrast, not weight.
- Never use `#FFFFFF` as background — always `--color-bg` (`#FFF5EF`).
- Never use border-radius above `25px` on non-pill elements.
- Never use the accent blue on body text — only on interactive elements.
- The dark section (`--color-bg-alt`) is for alternating contrast sections (e.g. featured work, contact CTA) — not the default.
